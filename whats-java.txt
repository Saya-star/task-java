Javaとは　課題

1. Java と C++ の違いを調べてみましょう

・Java
汎用性が高く、Webシステムやアプリに広く使われる
プラットフォームに依存しないので、どのような環境でも実行できる／プラットフォーム毎にプログラムを作り替える必要がない
大規模な開発に向いている

・C++
ポインタの概念がある（Javaではメモリを直接扱うことのリスクから操作できない）
プラットフォームごとにプログラムを作り替える必要がある
処理速度が速い
組み込みシステムの開発やロボット技術に活用される
ハードウェアと関係するシステム開発に特化しており、IoT開発に強みを持つ
Javaよりも低い階層で扱う関数が多く、開発したいプログラムに合わせて自分の好きなように環境を細かく設定できる


2. Java と JavaScript の違いを調べてみましょう

・Java
Webシステムやアプリ、業務システムなど大規模〜小規模な様々な分野で使われる
データ型を最初に決める形式の文法

・JavaScript
Webページ開発での使用がメイン（Webページに動作を与える）
データ型を適宜変更できる形式の文法（実行速度は遅いが、初心者にも書きやすい）


3. Java と Python の違いを調べてみましょう

・Java
大規模〜小規模な様々なシステムに利用される
データ型を最初に決める文法
ルールなど覚えなければならないことが多い（幅広い用途で使えることを意識して作られているため）

・Python
数値計算や統計処理に強く人工知能・機械学習にも使われる
データ型がプログラムが実行されるときに決まる
文法がシンプルで読みやすい・書きやすいため初心者にも向いている
